# CoreHR

**CoreHR** is a web-based application designed to efficiently manage employee information with a secure and user-friendly interface. This system allows administrators to manage users and their profiles while enforcing role-based access control.

## Features I Worked On
- **Login Module**: Secure login flow with session management and role-based access.
- **User Management**: Admin-only user registration and management through the dashboard.
- **User Profile**: View and edit profile functionality, including password reset.
- **Admin Dashboard**: Full control for administrators to manage employees and system settings.

> Note: There is no public registration. Only admins can register new users via the dashboard.

## Tech Stack
- **Frontend**: HTML, CSS, Bootstrap, JavaScript  
- **Backend**: Java Servlet  
- **Database**: MySQL  

## Highlights
- Role-based access control ensures that only authorized users can access certain features.
- Secure session handling for login and profile management.
- Modular design for user management and profile functionalities.
